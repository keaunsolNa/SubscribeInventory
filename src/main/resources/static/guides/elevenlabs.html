ElevenLabs 남은 글자 수(할당량) 확인 방법

ElevenLabs는 크레딧이 아니라 월 글자 수 할당량 방식입니다. 다행히 8개 서비스 중 조회가 가장 쉬운 편으로, 일반 키로 바로 확인됩니다. (2026-07 기준)

방법 1 — 콘솔에서 확인

elevenlabs.io 로그인 후 좌측 하단 프로필 → Subscription에서 사용량 바를 볼 수 있습니다.

방법 2 — API로 직접 조회

curl "https://api.elevenlabs.io/v1/user/subscription" \
  -H "xi-api-key: $ELEVENLABS_API_KEY"

응답에서 character_count(사용량), character_limit(플랜 한도), next_character_count_reset_unix(다음 리셋 시각)를 읽으면 됩니다. 남은 글자 수 = limit − count. 일반 API 키로 호출 가능해서 별도 관리자 키가 필요 없습니다.

방법 3 — 대시보드에서 다른 서비스와 한 번에

SubscribeInventory에서 한 화면으로 → ElevenLabs 잔여 글자 수를 게이지로 보여주고, OpenAI·Claude·xAI 등 8개 서비스와 함께 Slack 잔여량 알림까지 무료로 제공합니다. 키는 브라우저에만 저장됩니다.

자주 묻는 질문

특별한 키가 필요한가요?

아니요, 일반 API 키(xi-api-key 헤더)면 됩니다.

남은 글자 수는 어떻게 계산하나요?

character_limit − character_count이고, 리셋 시각은 next_character_count_reset_unix입니다.
