DeepSeek API 잔액 확인 방법

DeepSeek은 잔액 전용 API를 공식 제공하는 몇 안 되는 서비스라 조회가 간단합니다. (2026-07 기준)

방법 1 — 콘솔에서 확인

platform.deepseek.com 우측 상단에 잔액이 바로 표시되고, Billing 메뉴에서 상세 내역을 볼 수 있습니다.

방법 2 — API로 직접 조회

curl "https://api.deepseek.com/user/balance" \
  -H "Authorization: Bearer $DEEPSEEK_API_KEY"

응답의 balance_infos 배열에서 통화별 total_balance를 읽으면 됩니다. 일반 API 키로 호출 가능합니다. 잔액이 0이면 is_available: false이거나 배열이 비어 올 수 있으니 그 경우를 0으로 처리하세요.

방법 3 — 대시보드에서 다른 서비스와 한 번에

SubscribeInventory에서 한 화면으로 → DeepSeek 잔액과 함께 OpenAI·Claude·xAI 등 8개 서비스의 잔액·사용량을 무료로 한 번에 확인하고, Slack 잔여량 알림까지 받을 수 있습니다. 키는 브라우저에만 저장됩니다.
