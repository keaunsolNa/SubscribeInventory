OpenAI API 사용액 확인 방법

OpenAI는 잔여 크레딧을 API로 주지 않아서, "이번 달 얼마나 썼는지"를 추적하는 방식으로 관리합니다. 콘솔에서 보는 법과 API로 직접 조회하는 법을 정리했습니다. (2026-07 기준)

방법 1 — 콘솔에서 확인

platform.openai.com/usage에서 기간별 사용액을, Settings → Billing에서 선불 크레딧 잔액과 결제 수단을 볼 수 있습니다. 가장 정확하지만 매번 로그인해서 들어가야 합니다.

방법 2 — Costs API로 직접 조회

⚠️ 일반 API 키(sk-...)로는 안 됩니다. 조직 관리 화면에서 발급하는 Admin 키(sk-admin-...)가 필요하고, 키에 api.usage.read 권한이 있어야 합니다. (Settings → Organization → Admin keys)
curl "https://api.openai.com/v1/organization/costs?start_time=1751328000&limit=31" \
  -H "Authorization: Bearer $OPENAI_ADMIN_KEY"

start_time은 유닉스 타임(월초)이고, 응답의 버킷별 amount를 합치면 해당 기간 사용액(USD)이 됩니다. 참고로 이 엔드포인트는 가끔 10초 이상 걸리므로 읽기 타임아웃을 넉넉히(20초+) 잡는 게 안전합니다.

방법 3 — 대시보드에서 다른 서비스와 한 번에

SubscribeInventory에서 한 화면으로 → OpenAI 사용액과 함께 Anthropic·xAI·DeepSeek 등 8개 서비스의 잔액·사용량을 무료로 한 번에 확인하고, Slack 잔여량 알림까지 받을 수 있습니다. 키는 브라우저에만 저장됩니다.

자주 묻는 질문

일반 API 키(sk-...)로 사용액을 조회할 수 있나요?

안 됩니다. Costs API는 Admin 키(api.usage.read 권한) 전용이라 일반 추론용 키로는 401이 반환됩니다.

잔여 크레딧(잔액)을 API로 받을 수는 없나요?

공식 API로는 제공되지 않습니다. 선불 크레딧 잔액은 콘솔에서만 보이므로, API 기반 모니터링은 월 누적 사용액으로 하는 것이 현실적입니다.

Costs API 응답이 느린데 정상인가요?

네, 집계 특성상 가끔 10초 이상 걸립니다. 타임아웃을 20초 이상으로 설정하세요.
