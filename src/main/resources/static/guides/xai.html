xAI (Grok) 선불 크레딧 잔액 확인 방법

xAI는 잔액 확인이 유난히 까다롭습니다 — 이름만 보면 맞을 것 같은 prepaid/balance 엔드포인트가 콘솔과 다른 값을 주기 때문입니다. 실키로 검증한 올바른 방법을 정리했습니다. (2026-07 기준)

방법 1 — 콘솔에서 확인

console.x.ai → Billing → Credits에서 팀별 선불 크레딧 잔액을 볼 수 있습니다.

방법 2 — Management API로 직접 조회

⚠️ prepaid/balance를 쓰지 마세요. 이 엔드포인트는 충전 내역만 기록해서 사용액이 반영되지 않습니다("잔액이 안 줄어드는" 값). 콘솔과 일치하는 값은 인보이스 미리보기에서 계산해야 합니다.
curl "https://management-api.x.ai/v1/billing/teams/$TEAM_ID/postpaid/invoice/preview" \
  -H "Authorization: Bearer $XAI_MANAGEMENT_KEY"

응답의 coreInvoice.prepaidCredits(총 충전)와 coreInvoice.prepaidCreditsUsed(사용)를 읽습니다. 두 값 모두 센트 단위 문자열이고 음수(예: {"val":"-500"} = $5.00)라는 점에 주의하세요. 잔액 = |prepaidCredits| − |prepaidCreditsUsed| 입니다. 인증에는 추론 키가 아닌 Management API 키와 팀 ID(UUID)가 필요합니다.

방법 3 — 대시보드에서 다른 서비스와 한 번에

SubscribeInventory에서 한 화면으로 → 위 계산을 전부 대신해서 콘솔과 일치하는 xAI 잔액을 보여주고, OpenAI·Claude 등 8개 서비스와 함께 Slack 잔여량 알림까지 무료로 제공합니다. 키는 브라우저에만 저장됩니다.

자주 묻는 질문

prepaid/balance 값이 콘솔과 다른 이유가 뭔가요?

충전만 기록되고 사용이 반영되지 않는 원장이기 때문입니다. 인보이스 미리보기의 prepaidCredits/prepaidCreditsUsed로 계산해야 콘솔과 일치합니다.

어떤 키가 필요한가요?

Management API 키 + 팀 ID입니다. 추론용 API 키로는 빌링 조회가 안 됩니다.
