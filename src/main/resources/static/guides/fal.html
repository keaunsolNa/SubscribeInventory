fal.ai 크레딧 잔액 확인 방법

fal.ai 잔액 조회에서 가장 많이 막히는 지점은 403 오류입니다 — 키가 잘못된 게 아니라 키 스코프 문제입니다. 해결법까지 정리했습니다. (2026-07 기준)

방법 1 — 콘솔에서 확인

fal.ai/dashboard/billing에서 잔여 크레딧과 결제 내역을 볼 수 있습니다.

방법 2 — API로 직접 조회

⚠️ 기본 키로는 403이 납니다. fal.ai 키는 생성 시 API(기본, 추론 전용)와 ADMIN 두 스코프 중 고르게 되어 있고, 빌링 엔드포인트는 ADMIN 스코프 키 전용입니다. "This API key is not permitted to perform this action"이 뜨면 ADMIN 스코프로 재발급하세요.
curl "https://api.fal.ai/v1/account/billing?expand=credits" \
  -H "Authorization: Key $FAL_KEY_ID:$FAL_KEY_SECRET"

인증은 Bearer가 아닌 Key 스킴(key_id:key_secret 형식)을 씁니다. expand=credits를 붙이면 응답의 credits에서 잔액(USD)을 읽을 수 있습니다.

방법 3 — 대시보드에서 다른 서비스와 한 번에

SubscribeInventory에서 한 화면으로 → fal.ai 잔액과 함께 OpenAI·Claude·Stability 등 8개 서비스의 잔액·사용량을 무료로 한 번에 확인하고, Slack 잔여량 알림까지 받을 수 있습니다. 키는 브라우저에만 저장됩니다.

자주 묻는 질문

403 "not permitted to perform this action"이 떠요.

키 스코프 문제입니다. ADMIN 스코프로 새 키를 발급하면 해결됩니다.

인증 형식이 다른 서비스와 다른가요?

네, Authorization: Key key_id:key_secret 형식("Key" 스킴)입니다.
