Stability AI 크레딧 잔액 확인 방법

이미지 생성 서비스 Stability AI는 크레딧 방식이고, 잔액 API가 단순해서 조회가 쉽습니다. (2026-07 기준)

방법 1 — 콘솔에서 확인

platform.stability.ai → Account → Credits에서 잔액과 충전 메뉴를 볼 수 있습니다.

방법 2 — API로 직접 조회

curl "https://api.stability.ai/v1/user/balance" \
  -H "Authorization: Bearer $STABILITY_API_KEY"

응답이 {"credits": 25.0} 형태로 바로 옵니다. 일반 API 키로 호출 가능합니다. 크레딧은 자체 단위이며 구매 기준 1,000크레딧 = $10입니다.

방법 3 — 대시보드에서 다른 서비스와 한 번에

SubscribeInventory에서 한 화면으로 → Stability 크레딧과 함께 OpenAI·Claude·fal.ai 등 8개 서비스의 잔액·사용량을 무료로 한 번에 확인하고, Slack 잔여량 알림까지 받을 수 있습니다. 키는 브라우저에만 저장됩니다.
