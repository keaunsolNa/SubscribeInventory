Anthropic (Claude) API 사용액 확인 방법

Claude API의 비용 조회에는 일반 키가 아닌 조직(Organization) Admin 키가 필요합니다 — 개인 계정에서 이 메뉴가 안 보여 헤매는 경우가 많습니다. 순서대로 정리했습니다. (2026-07 기준)

방법 1 — 콘솔에서 확인

console.anthropic.com → Billing에서 크레딧 잔액과 사용 내역을 볼 수 있습니다.

방법 2 — Cost Report API로 직접 조회

⚠️ Admin 키(sk-ant-admin-...)는 조직 계정 전용입니다. 개인 계정이면 먼저 콘솔에서 조직을 구성해야 Admin Keys 메뉴가 나타납니다 (무료). 일반 키 (sk-ant-api...)로는 비용 API를 호출할 수 없습니다.
curl "https://api.anthropic.com/v1/organizations/cost_report?starting_at=2026-07-01T00:00:00Z" \
  -H "x-api-key: $ANTHROPIC_ADMIN_KEY" \
  -H "anthropic-version: 2023-06-01"

응답의 버킷별 금액을 합치면 기간 사용액(USD)입니다. 잔액 전용 API는 없으므로 월 누적 사용액 기준으로 모니터링합니다.

방법 3 — 대시보드에서 다른 서비스와 한 번에

SubscribeInventory에서 한 화면으로 → Claude 사용액과 함께 OpenAI·xAI·DeepSeek 등 8개 서비스의 잔액·사용량을 무료로 한 번에 확인하고, Slack 잔여량 알림까지 받을 수 있습니다. 키는 브라우저에만 저장됩니다.

자주 묻는 질문

일반 키(sk-ant-api...)로 사용액을 조회할 수 있나요?

안 됩니다. 비용 조회는 Admin 키 전용입니다.

개인 계정인데 Admin 키 발급 메뉴가 안 보여요.

Admin 키는 조직 계정 전용이라, 콘솔에서 조직을 구성하면 메뉴가 나타납니다.

잔여 크레딧을 API로 받을 수는 없나요?

잔액 전용 API는 없고 기간별 비용 리포트만 제공됩니다.
