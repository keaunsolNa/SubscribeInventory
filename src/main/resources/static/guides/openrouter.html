OpenRouter 크레딧 잔액 확인 방법

여러 LLM을 한 키로 쓰는 OpenRouter는 잔액 조회도 간단한 편입니다 — 다만 응답이 "잔액"이 아니라 "총 충전·총 사용" 형태라 한 번 계산이 필요합니다. (2026-07 기준)

방법 1 — 콘솔에서 확인

openrouter.ai/settings/credits에서 잔액과 충전 내역을 볼 수 있습니다.

방법 2 — API로 직접 조회

curl "https://openrouter.ai/api/v1/credits" \
  -H "Authorization: Bearer $OPENROUTER_API_KEY"

응답의 data.total_credits(총 충전)와 data.total_usage(총 사용)를 읽어서 잔액 = total_credits − total_usage 로 계산합니다 (USD). 일반 키(sk-or-...)로 호출 가능합니다.

방법 3 — 대시보드에서 다른 서비스와 한 번에

SubscribeInventory에서 한 화면으로 → OpenRouter 잔액을 계산까지 마친 값으로 보여주고, OpenAI·Claude·xAI 등 8개 서비스와 함께 Slack 잔여량 알림까지 무료로 제공합니다. 키는 브라우저에만 저장됩니다.

자주 묻는 질문

잔액은 어떻게 계산하나요?

total_credits − total_usage 입니다.

특별한 키가 필요한가요?

아니요, 일반 API 키면 됩니다.
