구독 서비스 대시보드

AI 구독형 서비스의 잔여 할당량 · 크레딧 · 이번 달 사용액을 한눈에

준비됨
🔑 내 API 키 (BYOK — 이 브라우저에만 저장)
키는 이 브라우저(localStorage)에만 저장됩니다. 조회할 때마다 요청에 실려 백엔드가 각 서비스로 중계만 하며, 서버에는 저장·로깅되지 않습니다. 비워두면 서버에 환경변수로 설정된 키(있다면)를 사용합니다.
🔔 Slack 알림 구독 (잔여량 경고)
구독하면 위 "내 API 키" 패널의 키와 Webhook URL이 서버에 암호화되어 저장되고(옵트인), 매시 정각 서버가 잔여량을 점검해 임계값 초과 시 입력한 Slack 채널로 알립니다. 구독 시 확인 메시지가 채널로 1회 발송되고, 같은 Webhook으로 다시 구독하면 설정이 갱신됩니다. 같은 경고 상태는 반복 발송하지 않으며, 해지하면 저장된 정보가 즉시 삭제됩니다.
